Pseudo-class
A special kind of selector that specifies the type or state. Usually this is some kind of qualitative sign: the reaction to the cursor pointing, the order of succession, and others.
Pseudo-element
Elements that do not exist in the HTML markup, but act as if you added a whole new HTML element to the markup. They are created and positioned exclusively using CSS. They are most often used to create various decorative elements (which do not carry meaningful meaning).
